# Kickoff Prompt

*Version: pending rewrite*

This file will contain the one-shot setup walkthrough your AI tool uses to stand up your Ellipsend-connected DM agent. It covers:

- Connecting to the Ellipsend MCP server
- Subscribing to webhooks and setting up your public endpoint
- Building a local dashboard with autopilot controls, allowlist UI, activity feed, and rating loop
- Securing your endpoint (webhook secret, rate limiting)
- Setting up a process manager so your agent survives restarts and reboots
- Configuring agent identity, voice, ICP, CTAs, labels, and escalation rules
- Running a synthetic smoke test before going live
