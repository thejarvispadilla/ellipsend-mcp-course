# Agent Playbook

*Version: pending rewrite*

This file is the persistent runtime config your AI tool loads every session. It defines:

- How the agent handles inbound DMs and comments
- When to escalate vs respond vs ignore
- Flag lifecycle (flagged, resolved, ignored)
- Security rules and prompt-injection defense
- Self-echo filter
- Debounce behavior
- Autopilot states (OFF, SAFE, ON)
